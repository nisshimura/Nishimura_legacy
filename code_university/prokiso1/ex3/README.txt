第3回授業内の例題・演習問題です。
zipファイルをダウンロード・解凍し、中のPythonコード (*.py) をSpyderで開いてください。

各ファイルのコード内上部のコメント行の例題・演習問題の内容に従い、
正しい結果が出力されるように、コードを修正してください。

演習問題について他の学生に教えてもらった場合は、その学生の名前を、
提出ボタン下の「限定公開コメントを追加」から必ず記入してください。
「教えてもらった学生: oooo, xxxx」といった形で記入してください。

修正したコードは指定された名前で新たに保存し、課題の添付ファイルとして提出してください。

提出するファイル (00000000を自身の学籍番号に置き換えること) :
ex3-1-00000000.py
ex3-2-00000000.py
ex3-3-00000000.py
