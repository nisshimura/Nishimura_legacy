pwd - 現在の作業ディレクトリの名前を出力する
ls
- ディレクトリの内容をリスト表示する
mkdir - ディレクトリを作成する
rmdir - 空のディレクトリを削除する
cp - ファイルやディレクトリのコピーを行う
rm - ファイルやディレクトリの削除を行う
mv - ファイルの移動 (や名前の変更) を行う
file - ファイルタイプを判定する
id - 現在のユーザ ID 名とグループ ID 名を表示する
groups - 現在のグループ名の表示
chmod - ファイルのモードビットを変更する
ln - ln 8.16 のマニュアルページ
ps - 現在実行されているプロセスのスナップショットを表示する
kill - プロセスを終了させる
